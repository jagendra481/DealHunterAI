# 🚀 DealHunterAI

An AI-powered affiliate price tracking platform built with Python and Flask.

## Features

- Live Amazon price tracking
- Telegram deal alerts
- Affiliate link generation
- Interactive web dashboard
- Product analytics
- Price history
- Scheduler
- SQLite database

## Tech Stack

- Python
- Flask
- SQLite
- Rainforest API
- Telegram Bot API
- Bootstrap 5
